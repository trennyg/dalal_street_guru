## Stock of the Week — Template

Use this exact format for all stock posts:

```
[COMPANY NAME] scores [X]/100 on our guru screener 🎯

✓ ROE: X% (Buffett approved ✓)
✓ EPS Growth 3Y: X%
✓ Operating Margin: X%
✓ Promoter Holding: X%
[✓ or ✗ one more relevant metric]

[Conviction tier: STRONG BUY / BUY / WATCH]

Full breakdown: stocks.relentlessais.com/[TICKER]

#NSE #IndianStocks #ValueInvesting #StockMarket
```

Rules:
- Only use stocks scoring 60+ (Buy tier or above)
- Always include the disclaimer in thread replies, not the main post
- Never promise returns or use "multibagger" without heavy caveats
- Add 2–3 sentence context: "Why this stock, why now?"

## YouTube Series: "Invest Like The Legends"

Each video follows this structure:
1. **Hook (0–30s):** What would [investor] think of [stock/sector]?
2. **Framework explanation (1–3 min):** How does [Buffett / RJ / Graham] evaluate stocks?
3. **Live screener demo (2–3 min):** Show stocks.ai filtering by that investor's profile
4. **3 stocks from the screener (3–5 min):** Walk through score breakdowns
5. **CTA (30s):** "Run this screen yourself at stocks.relentlessais.com"

Episode ideas:
- "What stocks would Rakesh Jhunjhunwala buy today?"
- "Warren Buffett's India portfolio — if he invested in NSE"
- "The Graham Value screen: finding cheap quality in Nifty 500"
- "SMILE framework — what Vijay Kedia looks for in small caps"
- "How mutual funds screen stocks — the Quality/MF filter explained"
